# Deferred-Authentication

Agent Skill: Deferred Authentication.

Authentication is not the goal; it is a capability. This skill helps agents keep long-running tasks focused on the original objective when login, OAuth, API keys, cookies, sessions, 401/403 errors, or publishing/deployment permissions block one step.

## Packages

- `deferred-authentication/`: English skill package.
- `deferred-authentication-zh/`: Chinese skill package.
- `deferred-authentication.zip`: Packaged English skill.
- `deferred-authentication-zh.zip`: Packaged Chinese skill.

## Core Idea

When a required capability blocks repeatedly, preserve the original goal by producing a resumable artifact instead of letting the task become a validation loop for that capability.
